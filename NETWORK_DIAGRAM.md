## Port Configuration

### Port 4200 (Frontend)
- **Protocol**: HTTP
- **Purpose**: Serves the Angular web application
- **Access**: Must be accessible from network
- **Firewall**: Must allow incoming TCP on 4200

### Port 8080 (Backend)
- **Protocol**: WebSocket (ws://)
- **Purpose**: Real-time game communication
- **Access**: Must be accessible from network
- **Firewall**: Must allow incoming TCP on 8080
